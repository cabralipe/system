Importar Oficinas

Selecione abaixo o arquivo Excel (.xlsx) contendo as informações das oficinas que deseja cadastrar de forma automática.

  • O arquivo deve conter as colunas obrigatórias: titulo, descricao, ministrante_id, vagas, carga_horaria, estado, cidade, datas, horarios_inicio, horarios_fim.
  • Para adicionar vários dias na mesma oficina, utilize vírgula para separar cada data e horário:
    Exemplo: datas = 01/05/2025,02/05/2025 | horarios_inicio = 08:00,09:00 | horarios_fim = 12:00,13:00
  • As listas "datas", "horarios_inicio" e "horarios_fim" devem ter a mesma quantidade de itens.
  • Informe o ID do ministrante (coluna ministrante_id) caso já exista no sistema.
